# Arpit Deshmukh - Portfolio V3

A modern, high-performance, dark-themed portfolio built with React and Tailwind CSS v4.

## 🚀 Tech Stack
- **Frontend:** React (Vite), Tailwind CSS v4, Framer Motion, Lucide React
- **Architecture:** Client-Server Monorepo
- **Deployment:** Vercel Ready

## 📂 Project Structure
- `/client` - React frontend application
- `/server` - Express backend (ready for API integration)

## 💻 Local Development

1. **Install dependencies for the frontend:**
   ```bash
   cd client
   npm install
   ```

2. **Run the development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   ```

## 🌐 Deployment (Vercel)
This project is configured to be easily deployable on Vercel.
1. Import the repository into Vercel.
2. Set the **Framework Preset** to `Vite`.
3. Set the **Root Directory** to `client`.
4. Vercel will automatically run `npm run build` and deploy the `dist` folder.

## ✨ Features
- Fully responsive modern design (Glassmorphism, CSS Grids).
- Interactive 3D hover effects on cards.
- Modular component structure.
- Smooth scroll animations via Framer Motion.
